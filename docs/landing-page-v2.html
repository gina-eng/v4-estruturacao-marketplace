Implementação em 5 Minutos

Simples, rápido e direto do seu terminal.

01

Instale o Claude Code

Interface de linha de comando segura da Anthropic.

npm install -g @anthropic-ai/claude-code
02

Adicione o Marketplace

Registre a fonte oficial dos plugins V4.

/plugin marketplace add guilhermelippert/v4-estruturacao-marketplace
03

Ative o Plugin

Baixe todas as skills de estruturação no seu workspace.

/plugin install v4-estruturacao-ia@v4-estruturacao-marketplace
