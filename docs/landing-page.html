Grafo de dependências

As skills se alimentam mutuamente. O resultado de uma etapa é insumo para a próxima.

O que é um checkpoint?

A cada entregável, a IA apresenta o resultado e pede sua validação antes de avançar.

IA
Aqui está a análise SWOT do seu cliente. Identifiquei 4 forças, 3 fraquezas, 5 oportunidades e 2 ameaças com base nos dados do V4MOS e no briefing.
Operador
Bom, mas ajusta o tom de voz pra mais informal. E adiciona a sazonalidade como oportunidade.
IA
Pronto! Ajustei o tom e adicionei sazonalidade como oportunidade. Posso avançar para o Posicionamento?
Como funciona o checkpoint

1. A IA gera o entregável com base em dados reais

2. Você revisa e pede ajustes

3. A IA refina até você aprovar

4. Só então o sistema avança

Nenhum entregável é gerado sem sua validação. Você está sempre no controle.

03 — Instalação

Pronto em 5 minutos

Siga o passo a passo abaixo. Cada comando é explicado para quem nunca usou terminal.

1

Instalar o Claude Code

Claude Code é uma ferramenta de IA que roda no seu terminal (linha de comando). Pense nele como um assistente inteligente que entende código e executa tarefas.

npm install -g @anthropic-ai/claude-code
Instala o Claude Code globalmente no seu computador. Precisa ter Node.js 18+ instalado.
2

Adicionar o marketplace

Registra o marketplace da V4 no seu Claude Code. Como adicionar uma "loja" de plugins.

/plugin marketplace add guilhermelippert/v4-estruturacao-marketplace
Execute esse comando dentro do Claude Code (após digitar claude no terminal).
3

Instalar o plugin

Baixa e ativa as 20+ skills de estruturação. Agora o Claude Code sabe fazer tudo do processo.

/plugin install v4-estruturacao-ia@v4-estruturacao-marketplace
Após instalar, digite / e veja as skills no autocomplete.
4

Criar workspace e configurar

Cria uma pasta de trabalho e configura tudo: diretórios, credenciais, gitignore.

mkdir ~/estruturacao-ia && cd ~/estruturacao-ia && claude

Depois, dentro do Claude Code:

/onboarding
O onboarding configura a estrutura de pastas, credenciais e o .gitignore automaticamente.
5

Ferramentas extras (opcional)

Algumas skills usam ferramentas externas. Instale conforme necessário:

Ferramenta Para que serve Instalação
Vercel CLI Deploy de landing pages npm i -g vercel
jq Processar JSON no terminal brew install jq
gh GitHub CLI para feedback brew install gh
6

Verificar instalação

Confirme que tudo está funcionando:

/plugin
Na aba "Installed" deve aparecer v4-estruturacao-ia. Digite / e as skills devem aparecer no autocomplete.
